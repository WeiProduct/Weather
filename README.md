# AI Weather Website

Static product website for AI Weather by WeiProduct.

Live site: https://weiproduct.github.io/Weather/

## Product Positioning

AI Weather helps users check current conditions, review forecasts, manage cities, and translate weather context into practical daily decisions.

## Pages

- `index.html` - Main product page
- `privacy-policy.html` - Privacy policy
- `terms.html` - Terms
- `support.html` - Support and contact

## Verification

```sh
node --check script.js
```
